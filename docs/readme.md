
# LabQuest

bem-vindo ao LabQuest, um sistema de gestão de qualidade gamificado. esta aplicação permite que organizações recompensem padrões de qualidade através de selos visuais, promovendo uma cultura de excelência.

## funcionalidades
- dashboard de usuário gamificado com galeria de selos.
- progressão de nível baseada em conquistas de qualidade.
- painel administrativo para gerenciar selos, usuários e empresas.
- atribuição de selos em lote via upload de excel.
- controle de acesso seguro baseado em funções usando supabase.
